Q. 画面収録の権限が必要なのはなぜですか？  
A. タイトルバーのタイトルの取得に画面収録の権限が必要です。  
Apple(Mac)がそう定めているので付与してあげてください。  
特に録画を残したりしているとかそういうことではありません。
