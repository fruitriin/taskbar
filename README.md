# セットアップの方法

### インストールする

[最新版のダウンロード](https://github.com/fruitriin/taskbar/releases)

### MacのDockを隠す

最初に起動したときはこのようにDockが邪魔なので、Dockを隠す必要があります。  
<img src="https://github.com/fruitriin/taskbar/blob/master/docs/setup-atfirst.png?raw=true" max-width=400px>

システム設定を開いて、Dockの設定を開いてください。
デスクトップとDockを開いてください。  
<img src="https://github.com/fruitriin/taskbar/blob/master/docs/find-dock-setting.png?raw=true"  max-width=400px>

Dockを自動的に表示/非表示にチェックをつけてください。
これでDockを普段を隠すことが可能です。  
<img src="https://github.com/fruitriin/taskbar/blob/master/docs/how-to-hide-dock.png?raw=true" max-width=400px>

Dockを小さくしたり、画面上の位置を左右いずれかに設定するとより邪魔にならないでしょう。  
<img src="https://github.com/fruitriin/taskbar/blob/master/docs/more-small-dock.png?raw=true" max-width=400px>

## FAQ

Q. 画面収録の権限が必要なのはなぜですか？  
A. タイトルバーのタイトルの取得に画面収録の権限が必要です。  
Apple(Mac)がそう定めているので付与してあげてください。  
特に録画を残したりしているとかそういうことではありません。

Q. Dockと場所がかぶってしまいました  
A. Dockの場所を移動しましょう（横とか）  
さらに最小サイズにして、普段は非表示にもしています。

タスクバーくんは強い！
